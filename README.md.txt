Career Finder: AI-powered Job Search Agent for Germany

An automated job serach and application intelligence agent designed to bridge the gap between software development and specialized SLAM/Robotics engineering.

Overview

This project automates the daily job hunting process by:
1. Scraping the latest roles for "SLAM", "Sensor Fusion" and "Robotics" across diverse industries in Germany.
2. Analyzing job descriptions against a professional CV to identify missing technical keywords and skill gaps.
3. Generating tailored cover letters and application "clues" using DeepSeek AI.
4. Tracking all potential applications in an automated Excel tracker with status monitoring.

Tech Stack

Language: Python 3.14+
AI Brain: DeepSeek AI (Open Source API)
Scraping: JobSpy (Multi-site job scraper)
Automation: GitHub Actions(CI/CD Pipeline)
Data Management: Pandas & OpenPyXL (Excel Automation)
